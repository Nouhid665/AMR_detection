The Challenge We Solve

Technical Challenge

  • DVB-S2X satellites use multiple modulation schemes (QPSK, BPSK, 16APSK, 32APSK) based on channel conditions
  • Ground receivers must identify the correct modulation to establish synchronization
  • Existing solutions are too complex, resource-intensive, and not future-proof
  • Need for real-time detection without manual intervention

Our AMR Solution

  • Novel soft-decision AMR algorithm with DVB-S2X compliance
  • Low computational complexity suitable for resource-constrained hardware
  • Single unified algorithm handles all current and future DVB-S2X waveforms
  • Real-time detection of modulation type, data rate, and signal parameters

Technical Capabilities

Modulation Support

QPSK, BPSK, 16APSK, 32APSK, and all DVB-S2X standard modulations

Real-Time Processing

Instant waveform detection and parameter estimation for live satellite feeds

Parameter Detection

Automatic recognition of data rate, code rate, and frame structure
